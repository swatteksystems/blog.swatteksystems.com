|                          | how-to.                  |                          |

|                          |     )                    |                          |
|                          |                          |                          |
|                          |     You will now create/ |                          |
|                          | pull the svn repo now:   |                          |
|                          |                          |                          |
|                          |     svn co httpss://m560 |                          |
|                          | x-driver.svn.sourceforge |                          |
|                          | .net/svnroot/m560x-drive |                          |
|                          | r m560x-driver ; cd m560 |                          |
|                          | x-driver/m560x/branches/ |                          |
|                          | m5602 ;                  |                          |
|                          |     make ; sudo make ins |                          |
|                          | tall ; modprobe m5602    |                          |
|                          |                          |                          |
